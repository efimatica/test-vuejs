## Efimatica JS/Vue test

La prueba técnica consiste en crear una página web que simule la acción de añadir uno o varios productos al carrito de compra.

Para realizar este ejercicio tienes a tú disposición una API con un listado de productos hábiles.

https://dummyjson.com/products
